# IP引擎 — 个人IP内容经营系统

> 不是 AI 写作工具，是帮你把内容变成长期资产的 IP 经营系统。
> # IP 引擎

帮个人创作者梳理 IP 定位、生成内容选题和成稿的 AI 工具。48h MVP。

## 解决什么问题

很多创作者感觉"自己有东西说，但说不清自己是谁"。
自由填写定位框，大多数人会卡住或写出无法用于 AI 个性化的模糊描述。

## 产品流程

引导式 Onboarding → IP 画像生成 → 选题建议 → AI 成稿 → 内容日历

用结构化问卷替代自由输入作为起点，
降低认知门槛，同时提升 AI 生成内容的个性化质量。

## 设计决策

内置 AI 质量评估机制：生成后对内容做自动评分，
低质量内容提示重新生成而不是直接交付给用户。

内置商业化路径分流：
根据创作者定位和受众规模，在内容日历中标注变现时机。

主动放弃的功能：UI 精细化 / 推荐评分展示 / 多人协作，
聚焦核心链路跑通。

## 技术栈

React + DeepSeek API + Supabase · Netlify 部署
